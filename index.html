What is RedMoon?

RedMoon (RMN) is a project that will bridge the divided world of NFT. This project aims to build the very foundation of the metaverse and will allow the connection of every single application of NFTs, from NFT artworks marketplace such as digital arts, paintings, photography, architecture, 3d model/sculpture to NFT games, DeFi, Dapps, Augmented Reality, Virtual Reality etc. having all of them in one place, creating a boundless ecosystem for the Web 3.0.

RedMoon is an all-in-one decentralized ecosystem for Web 3.0

How to buy RedMoon token?

Step 1: Buy BNB

Buy BNB in Binance.com, Crypto.com or any other exchange

Step 2: Send BNB to Wallet

Send BNB to Metamask or Trust wallet (Can be downloaded from Appstore/Playstore)

Step 3: Exchange BNB to SmartChain BNB

Easily done when viewing BNB balance

Step 4: Swap SmartChain BNB to RedMoon Token

Connect Trust wallet to PancakeSwap

Step 5: Hodl

Hold your token for profit
